# ASP.NET Core API Practice

## 개요

ASP.NET Core 백엔드 연습용 프로젝트입니다.

## 요구 환경

- .NET 9 SDK 이상
- 개발 중인 패키지: 확인: [aspCoreApiPractice.csproj](aspCoreApiPractice.csproj)

## 실행 방법

1. 복원, 빌드, 실행

```bash
dotnet restore
dotnet build
dotnet run --project [aspCoreApiPractice.csproj](http://_vscodecontentref_/12)
```
